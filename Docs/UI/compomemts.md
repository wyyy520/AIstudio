# AIStudio 通用组件设计规范

# 页面定位

**通用组件规范定义了 AIStudio 中所有可复用 UI 组件的设计规范。**

这些组件是组成 AIStudio 各个页面的基础构建块，必须在所有页面中保持一致的视觉风格和交互行为。

---

# 用户目标

| 用户 | 目标 |
|------|------|
| 前端开发 | 了解每个组件的设计规范，确保实现一致性 |
| 设计师 | 了解组件库边界，设计新页面时复用现有组件 |
| 产品经理 | 验收 UI 时对照组件规范检查一致性 |

# 1. 按钮 Button

### 尺寸

| 尺寸 | 高度 | 水平内边距 | 圆角 | 字号 | 图标大小 |
|------|------|-----------|------|------|----------|
| Large | 44px | 24px | 12px | 15px | 18px |
| Medium | 36px | 18px | 10px | 14px | 16px |
| Small | 28px | 12px | 8px | 13px | 14px |
| Mini | 22px | 8px | 6px | 12px | 12px |

### 类型

| 类型 | 深色主题 | 浅色主题 |
|------|----------|----------|
| Primary | `primary` 背景，白色文字 | `primary` 背景，白色文字 |
| Secondary | `bg-tertiary` 背景，`text-primary` 文字 | `bg-tertiary` 背景，`text-primary` 文字 |
| Outline | 透明背景，`border-default` 边框 | 透明背景，`border-default` 边框 |
| Ghost | 透明背景，悬停显示 `bg-hover` | 透明背景，悬停显示 `bg-hover` |
| Danger | `error` 背景，白色文字 | `error` 背景，白色文字 |
| Text | 无背景，`primary` 文字 | 无背景，`primary` 文字 |

### 状态

| 状态 | 视觉 |
|------|------|
| Default | 正常样式 |
| Hover | 背景加深/变亮 |
| Active | 进一步加深，轻微缩放（scale 0.97） |
| Focus | 焦点环（`primary` 外发光） |
| Disabled | 透明度 40%，禁止点击 |
| Loading | 文字前显示加载动画，禁止点击 |

### 图标按钮

- 正方形按钮
- 宽高等于对应尺寸高度
- 图标居中
- 带文字图标按钮：图标在左，文字在右，间距 6px

### 按钮组

- 并列按钮之间无间隙
- 中间按钮取消中间圆角
- 分割线在按钮之间

# 2. 输入框 Input

### 尺寸

| 尺寸 | 高度 | 内边距 | 圆角 | 字号 |
|------|------|--------|------|------|
| Large | 44px | 12px 16px | 10px | 14px |
| Medium | 36px | 10px 12px | 8px | 14px |
| Small | 28px | 8px 10px | 6px | 13px |

### 状态

| 状态 | 视觉 |
|------|------|
| Default | `bg-tertiary` 背景，`border-default` 边框 |
| Hover | 边框变 `border-strong` |
| Focus | 边框变 `primary`，外发光 `primary-bg` |
| Error | 边框变 `error`，外发光 `error-bg` |
| Disabled | 透明度 40%，禁止交互 |
| Readonly | 背景变 `bg-hover`，禁止编辑 |

### 变体

- **带前缀/后缀图标**：图标在输入框内左右两侧
- **带清除按钮**：右侧显示 X 清除按钮
- **密码输入**：右侧显示眼睛图标切换显示/隐藏
- **数字输入**：右侧显示上下箭头调节
- **多行文本**：最小高度 80px，高度自适应

### 特殊规则

- 聚焦时边框动画：150ms 颜色过渡
- 错误时抖动：300ms 轻微水平抖动

# 组件设计原则总结

1. **圆角统一**：所有组件使用圆角，无直角
2. **动画流畅**：所有交互有过渡动画，时长 150ms ~ 300ms
3. **状态完整**：每个组件都有 Default、Hover、Active、Focus、Disabled、Loading 状态
4. **深色优先**：默认深色主题，组件颜色以深色主题为基准
5. **尺寸统一**：使用统一的尺寸体系（Large / Medium / Small）
6. **颜色一致**：使用设计系统定义的颜色变量，不硬编码颜色

---

# 后续扩展

1. **更多组件**：随着产品发展，持续添加新组件
2. **组件变体**：根据实际使用需求，添加更多组件变体
3. **无障碍**：后续增加 ARIA 属性和屏幕阅读器支持
4. **国际化**：组件文本支持多语言
