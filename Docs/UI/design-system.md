# AIStudio 设计系统规范

# 页面定位

**设计系统规范是 AIStudio 所有 UI 组件和视觉设计的基础规范文档。**

它定义了：
- 整体设计哲学和设计原则
- 颜色系统（深色主题 + 浅色主题）
- 字体排版规范
- 间距规范
- 圆角规范
- 阴影规范
- 图标规范
- 基础组件设计规范
- 响应式适配规范

整个设计系统必须保持：
- **专业感**：企业级工程软件开发工具的专业气质
- **高级感**：极简现代，低饱和度，留白合理
- **科技感**：符合 AI 开发平台的定位
- **工程软件风格**：参考 VS Code、JetBrains IDE
- **长时间使用不疲劳**：低对比度，护眼设计

---

# 用户目标

1. **开发团队**：有统一的设计语言可以遵循，保证全软件 UI 风格一致
2. **前端开发**：可以直接根据设计系统生成组件代码
3. **设计师**：有明确的设计约束，避免设计发散
4. **最终用户**：获得一致、舒适、专业的使用体验

---

# 设计原则

## 必须遵守的核心原则

| 原则 | 优先级 | 说明 |
|------|--------|------|
| 专业桌面工具 | ★★★★★ | 不要做成网页，不要做成管理后台，要做成专业开发工具 |
| 圆角无处不在 | ★★★★★ | 所有元素都必须圆角，不允许直角出现 |
| 深色默认主题 | ★★★★★ | 默认使用深色主题，支持切换浅色主题 |
| 低饱和度配色 | ★★★★★ | 所有颜色饱和度要低，避免刺眼 |
| 合理留白 | ★★★★★ | 内容与内容之间要有足够留白 |
| 动画自然 | ★★★★★ | 所有交互动画要自然流畅，不突兀 |
| 统一图标风格 | ★★★★★ | 全部使用线性图标，不混用风格 |
| 长时间使用舒适 | ★★★★★ | 对比度适中，护眼设计 |

# 字体规范

## 字体家族

```css
/* 主字体（界面） */
--font-family-sans: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans", sans-serif;

/* 代码字体（代码编辑、日志） */
--font-family-mono: "Fira Code", "JetBrains Mono", "Consolas", "Monaco", "Courier New", monospace;
```

**要求：**
- 数字必须等宽：使用等宽字体或开启 font-feature-settings: "tnum"
- 代码区域必须使用独立的等宽字体
- 字体层级必须清晰

## 字体层级

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| `h1` | 24px | 600 | 32px | 页面大标题 |
| `h2` | 20px | 600 | 28px | 面板标题 |
| `h3` | 16px | 600 | 24px | 卡片标题 |
| `body` | 14px | 400 | 20px | 正文文本 |
| `body-sm` | 13px | 400 | 18px | 次要文本 |
| `caption` | 12px | 400 | 16px | 辅助说明、标签 |
| `code` | 13px | 400 | 18px | 行内代码 |
| `code-block` | 13px | 400 | 20px | 代码块 |

# 圆角规范

**整体要求：所有元素都必须圆角，不允许直角。**

| 元素类型 | 圆角半径 | 示例 |
|----------|----------|------|
| 大卡片 / 弹窗 | 16px | 工作流画布面板、主弹窗 |
| 中卡片 / 面板 | 12px | 侧边栏面板、属性面板 |
| 节点 / 按钮 | 10px | 工作流节点、普通按钮 |
| 输入框 / 菜单 | 8px | 文本输入框、下拉菜单 |
| 小标签 / 图标按钮 | 6px | 标签、小徽章 |

**通用规则：**
- 容器越大 → 圆角越大
- 容器越小 → 圆角越小
- 永远不要使用 0px（直角）

# 图标规范

## 风格要求

- **必须全部使用线性图标**
- 描边宽度统一：1.5px
- 圆角端点
- 不允许填充图标和线性图标混用
- 图标颜色跟随文本颜色

## 尺寸规范

| 使用场景 | 尺寸 |
|----------|------|
| 工具栏图标 | 20px |
| 侧边栏图标 | 18px |
| 节点内图标 | 16px |
| 按钮内图标 | 16px |
| 小标签图标 | 14px |

## 图标库

使用 [Lucide](https://lucide.dev/) 图标库，保持一致性。

# 动画规范

## 时长曲线

| 动画类型 | 时长 | 缓动函数 |
|----------|------|----------|
| 微交互（悬停） | 150ms | `ease-out` |
| 过渡（展开收起） | 200ms | `ease-in-out` |
| 模态弹窗 | 250ms | `cubic-bezier(0.4, 0, 0.2, 1)` |
| 页面切换 | 300ms | `cubic-bezier(0.4, 0, 0.2, 1)` |
| 数据流动动画 | 1500ms | `linear`（循环） |

## 动画规则

- **任何点击必须有动画反馈**
- **任何弹窗必须有过渡动画**
- **任何页面切换必须平滑**
- **鼠标悬停必须有反馈**
- **拖拽必须流畅**
- **缩放必须顺滑**

## 禁止

- 不允许闪烁动画
- 不允许快速变换颜色
- 不允许影响用户专注力的动画

# 基础组件规范

## 按钮

### 尺寸

| 尺寸 | 高度 | 内边距 | 圆角 | 字号 |
|------|------|--------|------|------|
| Large | 44px | 0 24px | 12px | 15px |
| Medium | 36px | 0 18px | 10px | 14px |
| Small | 28px | 0 12px | 8px | 13px |

### 类型

| 类型 | 样式说明 |
|------|----------|
| Primary | 主色背景，白色文字 |
| Secondary | 二级背景，主文字 |
| Outline | 透明背景，边框，主文字 |
| Ghost | 透明背景，文字，悬停显示背景 |
| Danger | 错误色背景，白色文字 |

### 状态

- **Default**：默认样式
- **Hover**：背景加深 10%，阴影增强
- **Active**：背景加深 20%，轻微缩放
- **Disabled**：透明度 40%，禁止点击

## 输入框

- 圆角：8px
- 边框：默认 `border-default`，聚焦 `primary`
- 内边距：上下 10px，左右 12px
- 高度：默认 36px
- 聚焦时有外发光：`0 0 0 3px var(--primary-bg)`

## 下拉菜单 / 选择框

- 圆角：8px
- 弹窗圆角：10px
- 选项悬停：`bg-hover`
- 选中项：`bg-active` + 主色左侧边条
- 动画：淡入 + 轻微向下滑动

## 卡片

- 圆角：根据尺寸 12px ~ 16px
- 背景：`bg-tertiary`
- 边框：可选细微边框 `border-subtle`
- 阴影：`shadow`
- 内边距：`spacing-4` (16px)

## 弹窗 / 模态框

- 圆角：16px
- 背景：`bg-secondary`
- 阴影：`shadow-xl`
- 动画：从中心轻微放大 + 淡入
- 背景遮罩：黑色 60% 透明度

## 侧边栏

- 背景：`bg-secondary`
- 宽度：默认 240px，可拖拽调整
- 边框：右侧 `border-subtle`
- 项目悬停：`bg-hover`
- 项目选中：`bg-active` + 左侧主色边条

## 分隔线

- 颜色：`border-subtle`
- 粗细：1px
- 不使用明显分割线，依靠背景色区分区域

# 交互规范

## 鼠标交互

| 操作 | 反馈 |
|------|------|
| 悬停 | 背景色变化，150ms 过渡 |
| 点击 | 轻微按下效果，立即反馈 |
| 拖拽 | 元素跟随鼠标，有轻微半透明 |
| 滚轮缩放 | 平滑缩放，以鼠标位置为中心 |

## 键盘交互

- 支持全键盘操作导航
- 快捷键遵循 VS Code 习惯
- Tab 键可聚焦所有可交互元素
- 聚焦时有可见的焦点环

## 滚动

- 使用平滑滚动
- 自定义滚动条样式，细宽度，低调颜色

---

# 响应式适配

因为是桌面应用，主要适配不同窗口尺寸：

| 窗口宽度 | 布局调整 |
|----------|----------|
| > 1200px | 三栏布局（左面板 + 画布 + 属性）全部可见 |
| 900px ~ 1200px | 左侧面板可折叠 |
| < 900px | 仅保留画布，面板可通过呼出按钮显示 |

---

# 主题切换

- 支持深色 / 浅色主题一键切换
- 切换时有平滑过渡动画
- 用户选择持久化存储
- 默认：深色主题
