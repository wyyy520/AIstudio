# AIStudio 设置页面设计规范

# 页面定位

**设置页面是用户配置 AIStudio 全局参数和个性化设置的中心页面。**

参考 **VS Code** 设置页面设计风格，提供：
- 设置分类导航
- 搜索过滤
- JSON 编辑模式
- 配置导入/导出
- 设置重置

---

# 用户目标

| 用户 | 目标 |
|------|------|
| 所有用户 | 自定义 AIStudio 的外观和行为 |
| 开发者 | 配置 API Key、环境变量、代理设置 |
| 高级用户 | 精细调整性能参数、快捷键绑定 |

# 页面结构

## 1. 顶部工具栏

**内容：**
- 左侧："设置" 标题
- 右侧：
  - 搜索框（搜索设置项）
  - 切换视图按钮（UI 模式 / JSON 模式）
  - 导入设置按钮
  - 导出设置按钮

**样式：**
- 高度：48px
- 背景：`bg-secondary`
- 边框底部：`border-subtle`

---

## 2. 左侧分类导航

**宽度：** 200px

**分类列表：**

| 分类 | 内容 |
|------|------|
| 外观 | 主题、字体、颜色、布局 |
| 编辑器 | 工作流编辑器、画布、节点 |
| 工作流 | 默认配置、运行参数、超时 |
| AI 助手 | API Key、模型选择、参数 |
| 环境 | Python、GPU、CUDA、路径 |
| 快捷键 | 快捷键绑定管理 |
| 插件 | 插件全局设置 |
| 账号 | 登录、同步、许可证 |
| 关于 | 版本、更新、反馈 |

**交互：**
- 点击分类切换右侧内容
- 搜索时高亮匹配的分类

---

## 3. 右侧设置内容

**默认视图：UI 模式**

按分类分组显示设置项：

**设置项结构：**
```
┌─────────────────────────────────────────────────┐
│  主题                                           │
│  选择 AIStudio 的显示主题                        │
│                                                 │
│  ○ 深色主题  ○ 浅色主题  ○ 跟随系统             │
│                                                 │
│  修改后立即生效                                  │
└─────────────────────────────────────────────────┘
```

**设置项元素：**
- 标题（h3）
- 描述文字（body-sm）
- 控件（根据类型）
- 重置按钮（悬停显示）

**控件类型：**

| 类型 | 控件 | 示例 |
|------|------|------|
| 枚举选择 | 单选按钮组 | 主题选择 |
| 布尔值 | 开关 | 自动保存 |
| 字符串 | 输入框 | 用户名 |
| 数字 | 数字输入框 | 字体大小 |
| 颜色 | 颜色选择器 | 自定义颜色 |
| 文件路径 | 路径选择器 | Python 路径 |
| 多选列表 | 复选框列表 | 启用的插件 |
| 键值对 | 键值编辑 | 环境变量 |
| 代码 | 代码编辑器 | 自定义 CSS |

**JSON 模式：**

切换到 JSON 模式时，显示完整的设置 JSON：
- 语法高亮
- 自动补全
- 错误提示
- 保存按钮

## 4. 分类详细说明

### 外观设置

- 主题：深色 / 浅色 / 跟随系统
- 主题色：主品牌色自定义
- 圆角大小：全局圆角调整
- 字体：界面字体、代码字体
- 字体大小：标准字号
- 图标大小：工具栏图标大小
- 布局密度：紧凑 / 标准 / 宽松
- 侧边栏位置：左侧 / 右侧
- 动画：启用/禁用动画

### 编辑器设置

- 网格大小
- 网格颜色
- 网格显示开关
- 节点默认大小
- 连线样式：贝塞尔 / 直线
- 自动保存间隔
- 自动备份
- 最大撤销步数
- 对齐辅助线开关
- 迷你地图开关

### 工作流设置

- 默认运行超时
- 最大并行节点数
- 失败重试次数
- 输出保留天数
- 自动清理临时文件
- 工作流存储路径

### AI 助手设置

- API 提供商选择
- API Key 配置
- 默认模型选择
- 温度参数
- 最大 Token
- 上下文窗口大小
- 流式输出开关

### 环境设置

- Python 解释器路径
- CUDA 路径
- 环境变量
- 代理设置
- 下载镜像源
- 工作区路径

### 快捷键设置

- 快捷键列表（表格形式）
  - 快捷键名称
  - 当前绑定
  - 修改按钮
- 搜索快捷键
- 重置快捷键
- 导入/导出快捷键配置

### 插件设置

- 已安装插件列表
- 每个插件的设置项
- 插件自动更新开关

### 账号设置

- 登录/注册
- 用户信息
- 许可证管理
- 数据同步设置
- 注销

### 关于

- 版本号
- 检查更新
- 更新日志
- 反馈问题
- 帮助文档
- 开源协议

# UI规范

- 参考 VS Code 设置页面
- 分类清晰，一目了然
- 设置项紧凑但不拥挤
- 每个设置项有说明文字
- 搜索实时过滤，精准定位
- UI 视图和 JSON 视图可切换

---

# 交互规范

| 操作 | 行为 |
|------|------|
| 修改设置 | 立即生效或显示"需重启"提示 |
| 搜索 | 实时过滤，高亮匹配 |
| 重置单设置 | 点击重置按钮，恢复默认值 |
| 重置所有 | 弹出确认对话框 |
| 导入设置 | 选择 JSON 文件导入 |
| 导出设置 | 下载 JSON 文件 |
| 快捷键修改 | 弹出录制弹窗，按新快捷键绑定 |

---

# 动画规范

| 动画 | 时长 | 说明 |
|------|------|------|
| 分类切换 | 200ms | 内容区域淡入淡出 |
| 搜索过滤 | 200ms | 设置项依次淡入/淡出 |
| 开关切换 | 150ms | 开关滑块滑动 |
| 模式切换 | 200ms | UI↔JSON 视图切换 |

# 状态设计

## 设置项状态

| 状态 | 表现 |
|------|------|
| 默认值 | 正常显示 |
| 已修改 | 左侧蓝色竖线标记 |
| 搜索匹配 | 黄色高亮背景 |
| 禁用 | 灰色，不可交互 |

---

# 错误状态

## 保存失败

- 提示错误信息
- 重试按钮

## 导入失败

- 提示 JSON 格式错误
- 显示具体错误位置

## API Key 验证失败

- 提示验证失败
- 显示错误原因

# 空页面设计

**搜索无结果时：**

- 居中显示："没有匹配的设置项"
- 建议："尝试其他搜索词"

---

# 加载状态

- 页面加载：骨架屏
- 设置保存：按钮加载状态
- 快捷键录制：等待输入状态

---

# 页面跳转关系

- **从仪表盘 →** 点击设置按钮进入
- **从工作流编辑器 →** 菜单 → 设置进入
- **从这里 →** 关闭返回前一个页面

---

# API交互

| API | 用途 |
|-----|------|
| `settings/get` | 获取所有设置 |
| `settings/set` | 更新设置项 |
| `settings/reset` | 重置设置 |
| `settings/export` | 导出设置 |
| `settings/import` | 导入设置 |
| `settings/validate` | 验证设置值 |
| `account/login` | 登录 |
| `account/logout` | 注销 |
| `update/check` | 检查更新 |

# 后续扩展

1. **设置同步**：登录后自动同步设置到云端
2. **设置预设**：预设多套设置方案（如"性能优先"、"开发模式"）
3. **插件级别的设置**：每个插件可以有独立的设置页面
4. **设置历史**：记录设置变更历史，可回滚
5. **工作区设置**：支持项目级别的设置覆盖
